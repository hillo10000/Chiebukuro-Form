
問題解決フォーム
このフォームを使用して、問題の詳細を入力し、出力結果を生成することができます。
出力結果は25文字ごとに改行され、項目ごとに空白行が追加されます。

使用方法

前提条件

インターネットブラウザ
テキストエディタ (推奨: VSCode, Sublime Text, Atom)



このフォームには以下の機能があります：

入力フィールド

タイトル、日付、客先名、機械の名称、周辺装置の機種名、自動プロの種類、発生状況、原因・理由、環境・注意点を入力できます。
ラベルボタン

各ラベルボタンをクリックして、出力結果の先頭に追加するラベルを選択できます。ラベルボタンをクリックすると、ボタンがアクティブ状態になり、そのラベルが出力結果の先頭に追加されます。もう一度クリックするとラベルが解除されます。
OKボタン

OKボタンをクリックすると、入力内容がまとめられ、出力結果に反映されます。出力結果は25文字ごとに改行され、各項目の間に空白行が追加されます。
コピー機能

コピー機能を使って、生成された出力結果をクリップボードにコピーできます。
ファイル保存機能

OKボタンをクリックすると、生成された出力結果をテキストファイルとして保存できます。ファイル名は自動的に生成されます。
